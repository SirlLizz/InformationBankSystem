# Information Bank System

## Задание на практику

- [X] Разработать информационную систему, включая следующие сущности и атрибуты:

    * Счет
        * Наименование
        * Валюта
        * Средства
    * Операция
        * Счёт
        * Тип
        * Сумма
        * Дата
        * Категория
        * Комментарий

- [ ] Создать экраны для работы.

- [ ] Требуется создать следующие роли в системе:

    - [ ] Просмотр счетов (позволяет просматривать счета и операции)
    - [ ] Редактор операций (позволяет просматривать счета и операции, а также редактировать и создавать новые операции на счете)
    - [ ] Редактор счетов (позволяет просматривать и редактировать счета и операции)
    - [ ] Также требуются создать роль уровня строк, позволяющую пользователю работать с архивными счетами (остальные пользователи не должны видеть архивные счета).
